## 第一步：购买 .cn 域名 🛒

### 推荐平台（国内）

1. **阿里云万网** (推荐)
   - 网址：https://wanwang.aliyun.com
   - 价格：首年约 29-59 元（根据域名不同）
   - 特点：服务稳定，支持支付宝

2. **腾讯云**
   - 网址：https://dnspod.cloud.tencent.com
   - 价格：首年约 28-58 元
   - 特点：与微信生态集成好

3. **百度云**
   - 网址：https://cloud.baidu.com/product/bcd.html
   - 价格：首年约 29 元起

### 购买步骤（以阿里云为例）

1. 打开 https://wanwang.aliyun.com
2. 点击"域名注册"或"域名查询"
3. 输入想要的域名（例如：`guanting.cn`）
4. 选择未被注册的域名，点击"立即购买"
5. 填写信息，使用支付宝完成支付

### 域名命名建议
- 简短易记（3-8 个字符）
- 拼音或英文组合
- 避免使用商标或品牌词

## 第二步：部署网站到 Vercel ☁️

### 什么是 Vercel？
- 免费的静态网站托管平台
- 自动 HTTPS
- 全球 CDN 加速
- 国内访问速度一般，但稳定

### 部署步骤

1. **注册 Vercel 账号**
   - 网址：https://vercel.com
   - 可以用 GitHub、Google 或邮箱注册

2. **上传网站**
   - 方式一：使用 Vercel CLI（我帮你执行）
   - 方式二：推送到 GitHub，Vercel 自动部署

3. **绑定域名**
   - 在 Vercel 项目设置中添加你的域名
   - Vercel 会给你两个 DNS 记录：
     ```
     记录类型: CNAME
     主机记录: @
     记录值: cname.vercel-dns.com
     ```

---

## 第三步：域名解析 🌐

### 在阿里云配置 DNS

1. 登录阿里云控制台
2. 进入"域名" → 找到你的域名
3. 点击"解析设置"
4. 添加以下记录：

   ```
   记录类型: CNAME
   主机记录: @
   记录值: cname.vercel-dns.com
   TTL: 10分钟
   ```

5. 保存后等待生效（通常 5-10 分钟）

## 第四步：验证 ✅

### 检查部署

2. 应该能看到"👋 欢迎来到我的网站"页面
3. 在手机上打开测试移动端适配
